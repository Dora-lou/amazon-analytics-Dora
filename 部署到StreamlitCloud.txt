部署到 Streamlit Cloud（快速上线）
==============================

目标：让同事在 Windows / Mac 都能直接用浏览器访问，无需安装 Python。

一、准备 GitHub 仓库
------------------
1) 注册/登录 GitHub。
2) 新建一个仓库（例如：amazon-analytics-dora）。
3) 把整个 amazon-analytics 文件夹中的这些文件上传到仓库根目录：
   - app.py
   - requirements.txt
   - README.md
   - 使用说明.txt（可选）
   - data/（可选；如果里面有敏感数据不要上传）

注意：
- 不要上传 .venv/、__pycache__/、dist/、你的原始报表文件（含业务数据）。

二、在 Streamlit Cloud 创建应用
----------------------------
1) 打开：https://share.streamlit.io
2) 用 GitHub 登录。
3) 点击 New app
4) 选择：
   - Repository：你刚创建的仓库
   - Branch：main（或你的分支）
   - Main file path：app.py
5) 点击 Deploy，等待部署完成。

三、分享给同事
------------
部署完成后会得到一个链接（形如：https://xxx.streamlit.app）
把这个链接发给同事即可，Windows / Mac 都能打开使用。

四、重要说明（云端存储差异）
------------------------
Streamlit Cloud 会重启/更新应用，本地写入文件不保证长期保存。
因此本项目在云端会自动改为：
- “保存历史数据 / 规则模板” 默认保存在当前会话内（session_state），避免写文件失败导致页面断开。
- 如果你需要长期保存（跨多人/跨天），建议后续接数据库（例如 Supabase/SQLite/云存储）。
